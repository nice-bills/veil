Invisible Wallet - Vanilla JS Demo

This demo shows how to use the Invisible Wallet SDK without any framework dependencies.

1. Register

Create a new passkey credential and compute your wallet address.

2. Deploy

Deploy your wallet contract on-chain (requires a fee-paying keypair).

3. Login

Restore your wallet session from localStorage.

4. Sign Test

Sign a test payload with your passkey.
